IMPORTANT NOTE ABOUT APOTEK EXTRAWIDE FONT

The Apotek ExtraWide font is a commercial font and needs to be purchased or licensed properly.
You'll need to place the following files in this directory:
- ApotekExtraWide-Semibold.woff2
- ApotekExtraWide-Semibold.woff

You can purchase or license this font from the foundry that created it (Fontfabric) or from services like MyFonts.

Once you have the font files, place them in this directory for the website to use them properly.

If you don't have the font files, the website will fall back to Arial.
